# 此刻地球 · 地球正念观想

一个可直接运行的网页 MVP：根据当前时间计算地球昼夜分布，从用户所在位置观看地球，并完成 3、7、12 分钟的原创正念观想。

## 已实现

- 当前日期与时间驱动的昼夜地球
- WebGL 3D 地球、拖动与缩放；不支持 WebGL 时自动切换二维备用视图
- GPS 大致位置、城市预设、手动经纬度
- 3 分钟「回到此刻」、7 分钟「拥抱地球」、12 分钟「和平观想」
- 引导文字、设备中文语音、程序化环境声音与钟声
- 暂停、继续、提前结束、自然完成
- 完成后的感受记录与每周统计
- 星期三提醒设置（网页打开或后台仍可用时）
- PWA 清单、离线缓存与安装入口
- 本地存储；定位与练习记录不会上传服务器
- 响应式桌面与手机布局、减少动态效果、键盘焦点样式

## 最快使用方式

直接打开：

```text
dist/index.html
```

该文件已经把脚本、样式和地球纹理内联，可离线打开。浏览器对 `file://` 页面的位置、通知和 PWA 安装可能有限制；完整体验建议通过本机服务器或 HTTPS 网站打开。

## 本地开发

需要 Node.js 20 或更高版本。

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run typecheck
npm run build
```

本地预览：

```bash
npm run preview
```

## 部署

将 `dist/` 目录中的全部文件上传到任意静态网站托管服务即可。站点使用相对路径，适合部署在根目录或子目录。

为了使用定位、系统通知、Service Worker 与 PWA 安装，线上地址应启用 HTTPS。

## 原生 App 下一步

当前交付物是网页/PWA 版本。浏览器无法保证在网页完全关闭时准时触发每周提醒。要获得 iOS/Android 原生定时通知，可将本项目用 Capacitor 封装，并接入 Local Notifications；现有 UI、地球渲染、记录和冥想逻辑可继续复用。

## 测试入口

开发测试时在地址后加入：

```text
?demo=fast
```

练习时长会按约 10 倍速度运行，便于测试自然完成、感受记录和统计流程。正常用户地址不要添加该参数。

## 数据与隐私

- GPS 仅在用户主动点击时调用。
- 经纬度、偏好和练习记录仅写入浏览器本地存储。
- 项目没有后端、账号系统、分析 SDK 或广告 SDK。
- 语音由设备的 Web Speech API 提供；环境声由 Web Audio API 实时生成。
